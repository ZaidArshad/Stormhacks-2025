## How to run
Have python3 installed, we used python 3.13

### Create virtual enviornment
`python -m venv myenv`

Windows: `myenv\Scripts\activate.bat`

Mac/Linux: `source myenv/bin/activate` 

`pip install -r requirements.txt`
